## 📦 Installation Guide

1. Clone the repository:
```bash
git clone https://github.com/DhananjaiPS/amazonReactClone.git
cd amazonReactClone
```

2. Install dependencies:
```bash
npm install
```

3. Configure environment:
```bash
cp .env.example .env
```

4. Start development server:
```bash
npm run dev
```
